# Desenho em JavaScript

Projeto simples de um canvas interativo feito em JavaScript.  (inspirado no paint)

## Funcionalidades atuais
- Desenhar com o mouse
- Escolher cor do pincel
- Ajustar tamanho do pincel
- Limpar o canvas
- Utilizar a borracha

## Como usar
1. Abrir `index.html` no navegador
2. Usar os controles para desenhar e limpar

## Melhorias recentes
- Layout reorganizado, funções e botões à esquerda, para melhor vaisibilidade e funcionalidade
- Canvas fixo, evitando a rolagem
- Inserção da Borracha
